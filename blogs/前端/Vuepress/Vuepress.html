    # 1. Vuepress

    Vue 驱动的静态网站生成器。

    Vuepress 是 Vuejs 官方提供的一个是Vue驱动的静态网站生成器，基于Markdown语法生成网页。 官网 (opens new window)

    # 1.1 Vuepress搭建

    CSDN参考文档 (opens new window)

    #1.新建一个文件夹，取名webBlog，用于项目名称，并进入

#2.安装本地依赖 
yarn add -D vuepress

#3. 初始化项目， 会生成一个package.json文件
yarn init -y 

#4. 创建文档
mkdir docs 
echo '# Hello VuePress' > docs/README.md

#5. package.json 中添加一些 scripts
{
  "scripts": {
    "docs:dev": "vuepress dev docs",
    "docs:build": "vuepress build docs"
  }
}

#6. 本地启动
yarn docs:dev

    # 3. 上传github 使用github Pages

    GitHub Pages 文档 (opens new window)

    个人博客地址 (opens new window)

    1. 新建一个仓库，仓库的名称以github用户名命名。

      即: 输入 username.github.io 作为存储库名称。 将 username 替换为你的 GitHub 用户名。 例如，如果用户名为 octocat，则存储库名称应为 octocat.github.io。

    2. 将npm run build命令生成的打包文件夹(public)上传到该仓库下。

      注意: 只有将生成的打包文件夹传上去才可以。而不是吧整个项目上传。所以只需要把public文件夹上传即可，但是每次打包都会吧.git文件夹删掉，所以每次打包都会丢失git。所以使用脚本上传。

    3. 配置该仓库，选择指定分支(这边选择master分支)，点击save即可。

    4. 配置完成后，即可使用https://www.Z001Shen.github.io 访问。(提交代码后 可能需要等待一会)

    # 3.1 使用脚本上传打包文件至github

    在文件跟目录新建 deploy.sh

    #!/usr/bin/env sh
 
# 确保脚本抛出遇到的错误
set -e
 
# 生成静态文件， npm run build
npm run build

# # 进入生成的文件夹
cd ./public

# git初始化，每次初始化不影响推送
git init
git add -A
git commit -m 'deploy'
git branch -M master

# 推送到git master分支
git push -f https://github.com/Z001Shen/Z001Shen.github.io.git master
echo "部署成功"
